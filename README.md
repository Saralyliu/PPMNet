# PPMNet
PPMNet: Adaptive Prototype Merging with CLIP-Generated Paired Pseudo-Masks for Weakly-Supervised Few-Shot Segmentation
# Requirements
Python 3.9
PyTorch 1.13
cuda 11.7
tensorboard 2.18
# Datasets
PASCAL-5i: VOC2012 + SBD
COCO-20i: COCO2014
Please see OSLSM and FWB for more details on datasets.

This is the official implementation of the paper 'PPMNet: Adaptive Prototype Merging with CLIP-Generated Paired Pseudo-Masks for Weakly-Supervised Few-Shot Segmentation' submitted to The Visual Computer. If you use this code or find our work helpful, please consider citing our paper.
